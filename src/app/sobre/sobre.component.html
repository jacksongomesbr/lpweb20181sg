Sobre

Este software web fornece as seguintes funcionalidades:

  • Cadastrar partida
  • Dar lance e validar lance de placar
  • Dar lance e validar lance de quantidade de gols de um jogador
  • Apresentar estatísticas de lances
Aviso: Este software é fornecido como demonstração prática de recursos do framework Angular e não deve ser considerado como funcional e nem livre de erros. Entretanto, pode servir como base para outros projetos e trabalhos de sala de aula.
